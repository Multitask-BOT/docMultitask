# 🎟️ Autorôles par invitation

Attribue automatiquement des rôles selon **le lien d’invitation** utilisé par le membre qui rejoint.

## Configuration

### `/config autoroles-invitation ajouter lien:CODE role:@Role`

Associe un **CODE** d’invitation à un rôle.

### `/config autoroles-invitation supprimer lien:CODE`

Retire l’association.

### `/config autoroles-invitation liste`

Affiche les mappages actifs.

> Le bot doit pouvoir **lire et synchroniser** les invitations du serveur.
