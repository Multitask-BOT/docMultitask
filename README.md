## Comment utiliser cette doc
- Le menu *Module* regroupe les fonctionnalités par thème.
- La section *Utils → Mises à jour* récapitule l'historique des versions.
- Les exemples utilisent la syntaxe *slash commands* de Discord (ex: `/ban @user raison:`).

> Astuce : sur GitBook, **Ctrl + K** permet d’ouvrir la recherche.
