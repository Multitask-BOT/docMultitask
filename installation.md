# ⚙️ Installation

Processus d'ajout et de configuration de MultitaskBOT sur votre serveur.

## Inviter le bot
1. Ouvrez le site : **https://www.multitask.bebech.fr**
2. Cliquez sur **Ajoute-moi** puis connectez-vous si besoin.
3. Sélectionnez votre serveur puis **Continuer**.
4. Validez les permissions recommandées, puis **Autoriser**.

> Vous pourrez ajuster finement les permissions par salon ensuite (Discord → Paramètres du serveur → Rôles/Permissions).

## Pré-requis recommandés
- Rôle dédié au bot avec les permissions de **modération** et de **gestion des messages**.
- Activer les *historique des messages* si vous utilisez la purge.
- Pour les **vocaux privés**, le bot doit avoir *Déplacer des membres*.

## Vérification
Utilisez `/info-bot` pour vérifier la présence et la latence du bot.
